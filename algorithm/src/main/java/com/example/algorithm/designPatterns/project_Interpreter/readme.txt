描述
每个乘车系统都会有相应的付费提示，针对不同的人群，会有不同提示。乘客刷卡后会提供的信息包含 3 个部分，"城市, 类别, 乘车卡(可选)"。
目前有 2 条乘车提示：

免费乘车：
本次乘车免费~
付费乘车：
请付费！
规则定义有两种人群是可以免费乘车的：

孕妇、儿童、残疾人、军人 人群；
城市为 杭州 或 绍兴，且类别是 老人 或 学生，并持有 市民卡 或 交通卡 的人群。
expression 包说明：

Expression：表达式接口
InterpreterPattern：规则定义类
po 包说明：
AndExpression：具体表达式，逻辑规则 与
OrExpression：具体表达式，逻辑规则 或
TerminalExpression：终结符表达式
Main 类中会读取乘客的信息，通过付费系统返回的提示进行提示。请实现相应的类，完成付费提示系统。

样例
样例一
输入：

杭州, 老人, 交通卡
输出：

本次乘车免费~
样例二
输入：

杭州, 妇女
输出：

请付费！
